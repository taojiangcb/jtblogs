жөҸи§ҲеҷЁеҺҹзҗҶ3

DOM ж ‘

д»ҺвҪ№з»ңдј з»ҷжёІжҹ“еј•ж“Һзҡ„ HTML вҪӮд»¶еӯ—иҠӮжөҒжҳҜвҪҶжі•зӣҙжҺҘиў«жёІжҹ“еј•ж“ҺзҗҶи§Јзҡ„пјҢжүҖд»ҘиҰҒе°Ҷе…¶иҪ¬еҢ–дёәжёІжҹ“еј•ж“Һ
иғҪеӨҹзҗҶи§Јзҡ„еҶ…йғЁз»“жһ„пјҢиҝҷдёӘз»“жһ„е°ұжҳҜ DOMгҖӮ

  • д»Һв»ҡвҫҜзҡ„и§Ҷв»ҶжқҘзңӢпјҢDOM жҳҜвҪЈжҲҗв»ҡвҫҜзҡ„еҹәзЎҖж•°жҚ®з»“жһ„гҖӮ
  • д»Һ JavaScript и„ҡжң¬и§Ҷв»ҶжқҘзңӢпјҢDOM жҸҗдҫӣз»ҷ JavaScript и„ҡжң¬ж“ҚдҪңзҡ„жҺҘвјқпјҢйҖҡиҝҮиҝҷеҘ—жҺҘвјқпјҢ
    JavaScript еҸҜд»ҘеҜ№ DOM з»“жһ„иҝӣвҫҸи®ҝй—®пјҢд»ҺвҪҪж”№еҸҳвҪӮжЎЈзҡ„з»“жһ„гҖҒж ·ејҸе’ҢеҶ…е®№гҖӮ
  • д»Һе®үе…Ёи§Ҷв»ҶжқҘзңӢпјҢDOM жҳҜвјҖйҒ“е®үе…ЁйҳІжҠӨзәҝпјҢвјҖдәӣдёҚе®үе…Ёзҡ„еҶ…е®№еңЁ DOM и§Јжһҗйҳ¶ж®өе°ұиў«жӢ’д№Ӣв»”еӨ–дәҶгҖӮ

DOM ж ‘

  • HTML и§ЈжһҗеҷЁпјҲHTMLParserпјү
  • вҪ№з»ңиҝӣзЁӢеҠ иҪҪдәҶеӨҡе°‘ж•°жҚ®пјҢHTML и§ЈжһҗеҷЁдҫҝи§ЈжһҗеӨҡе°‘ж•°жҚ®гҖӮ
  • вҪ№з»ңиҝӣзЁӢжҺҘж”¶еҲ°е“Қеә”еӨҙд№ӢеҗҺпјҢдјҡж №жҚ®е“Қеә”еӨҙдёӯзҡ„ content-type еӯ—ж®өжқҘеҲӨж–ӯвҪӮд»¶зҡ„зұ»еһӢпјҢвҪҗеҰӮ contenttype
    зҡ„еҖјжҳҜвҖңtext/htmlвҖқпјҢйӮЈд№ҲжөҸи§ҲеҷЁе°ұдјҡеҲӨж–ӯиҝҷжҳҜвјҖдёӘ HTML зұ»еһӢзҡ„вҪӮд»¶пјҢз„¶еҗҺдёәиҜҘиҜ·жұӮйҖүжӢ©жҲ–
    иҖ…еҲӣе»әвјҖдёӘжёІжҹ“иҝӣзЁӢгҖӮжёІжҹ“иҝӣзЁӢеҮҶеӨҮеҘҪд№ӢеҗҺпјҢвҪ№з»ңиҝӣзЁӢе’ҢжёІжҹ“иҝӣзЁӢд№Ӣй—ҙдјҡе»әвҪҙвјҖдёӘе…ұдә«ж•°жҚ®зҡ„з®ЎйҒ“пјҢ
    вҪ№з»ңиҝӣзЁӢжҺҘж”¶еҲ°ж•°жҚ®еҗҺе°ұеҫҖиҝҷдёӘз®ЎйҒ“вҫҘвҫҜж”ҫпјҢвҪҪжёІжҹ“иҝӣзЁӢеҲҷд»Һз®ЎйҒ“зҡ„еҸҰеӨ–вјҖз«ҜдёҚж–ӯең°иҜ»еҸ–ж•°жҚ®пјҢе№¶еҗҢ
    ж—¶е°ҶиҜ»еҸ–зҡ„ж•°жҚ®вҖңе–ӮвҖқз»ҷ HTML и§ЈжһҗеҷЁгҖӮдҪ еҸҜд»ҘжҠҠиҝҷдёӘз®ЎйҒ“жғіиұЎжҲҗвјҖдёӘвҖңвҪ”з®ЎвҖқпјҢвҪ№з»ңиҝӣзЁӢжҺҘж”¶еҲ°зҡ„
    еӯ—иҠӮжөҒеғҸвҪ”вјҖж ·еҖ’иҝӣиҝҷдёӘвҖңвҪ”з®ЎвҖқпјҢвҪҪвҖңвҪ”з®ЎвҖқзҡ„еҸҰеӨ–вјҖз«ҜжҳҜжёІжҹ“иҝӣзЁӢзҡ„ HTML и§ЈжһҗеҷЁпјҢе®ғдјҡеҠЁжҖҒжҺҘ
    ж”¶еӯ—иҠӮжөҒпјҢе№¶е°Ҷе…¶и§Јжһҗдёә DOMгҖӮ

  1. йҖҡиҝҮеҲҶиҜҚеҷЁе°Ҷеӯ—иҠӮжөҒиҪ¬жҚўдёә TokenгҖӮ
  2. з¬¬вјҶдёӘе’Ңз¬¬дёүдёӘйҳ¶ж®өжҳҜеҗҢжӯҘиҝӣвҫҸзҡ„пјҢйңҖиҰҒе°Ҷ Token и§Јжһҗдёә DOM иҠӮзӮ№пјҢе№¶е°Ҷ DOM иҠӮзӮ№ж·»еҠ еҲ° DOM
    ж ‘дёӯгҖӮ

HTML и§ЈжһҗеҷЁз»ҙжҠӨдәҶвјҖдёӘ Token ж Ҳз»“жһ„пјҢиҜҘ Token ж Ҳдё»иҰҒвҪӨжқҘи®Ўз®—иҠӮзӮ№д№Ӣй—ҙзҡ„вҪ—вјҰе…ізі»пјҢеңЁз¬¬вјҖдёӘйҳ¶
ж®өдёӯвҪЈжҲҗзҡ„ Token дјҡиў«жҢүз…§йЎәеәҸеҺӢеҲ°иҝҷдёӘж ҲдёӯгҖӮ

  • еҰӮжһңеҺӢвјҠеҲ°ж Ҳдёӯзҡ„жҳҜ StartTag TokenпјҢHTML и§ЈжһҗеҷЁдјҡдёәиҜҘ Token еҲӣе»әвјҖдёӘ DOM иҠӮзӮ№пјҢз„¶еҗҺе°ҶиҜҘ
    иҠӮзӮ№еҠ вјҠеҲ° DOM ж ‘дёӯпјҢе®ғзҡ„вҪ—иҠӮзӮ№е°ұжҳҜж ҲдёӯзӣёйӮ»зҡ„йӮЈдёӘе…ғзҙ вҪЈжҲҗзҡ„иҠӮзӮ№гҖӮ

  • еҰӮжһңеҲҶиҜҚеҷЁи§ЈжһҗеҮәжқҘжҳҜвҪӮжң¬ TokenпјҢйӮЈд№ҲдјҡвҪЈжҲҗвјҖдёӘвҪӮжң¬иҠӮзӮ№пјҢз„¶еҗҺе°ҶиҜҘиҠӮзӮ№еҠ вјҠеҲ° DOM ж ‘дёӯпјҢ
    вҪӮжң¬ Token жҳҜдёҚйңҖиҰҒеҺӢвјҠеҲ°ж ҲдёӯпјҢе®ғзҡ„вҪ—иҠӮзӮ№е°ұжҳҜеҪ“еүҚж ҲйЎ¶ Token жүҖеҜ№еә”зҡ„ DOM иҠӮзӮ№гҖӮ

  • еҰӮжһңеҲҶиҜҚеҷЁи§ЈжһҗеҮәжқҘзҡ„жҳҜ EndTag ж ҮзӯҫпјҢвҪҗеҰӮжҳҜ EndTag divпјҢHTML и§ЈжһҗеҷЁдјҡжҹҘзңӢ Token ж ҲйЎ¶зҡ„
    е…ғзҙ жҳҜеҗҰжҳҜ StarTag divпјҢеҰӮжһңжҳҜпјҢе°ұе°Ҷ StartTag div д»Һж Ҳдёӯеј№еҮәпјҢиЎЁвҪ°иҜҘ div е…ғзҙ и§Јжһҗе®ҢжҲҗгҖӮ

HTML и§ЈжһҗеҷЁејҖе§ӢвјҜдҪңж—¶пјҢдјҡй»ҳи®ӨеҲӣе»әдәҶвјҖдёӘж №дёә document зҡ„з©ә DOM з»“жһ„





еңЁе®һйҷ…вҪЈдә§зҺҜеўғдёӯпјҢHTML жәҗвҪӮд»¶дёӯж—ўеҢ…еҗ« CSS е’Ң JavaScriptпјҢвјңеҢ…еҗ«еӣҫвҪҡгҖҒвҫійў‘гҖҒи§Ҷйў‘зӯүвҪӮд»¶пјҢжүҖ
д»ҘеӨ„зҗҶиҝҮзЁӢиҝңвҪҗдёҠвҫҜиҝҷдёӘвҪ°иҢғ Demo еӨҚжқӮгҖӮ

JavaScript жҳҜеҰӮдҪ•еҪұе“Қ DOM вҪЈжҲҗзҡ„

  • еҪ“и§ЈжһҗеҲ° еҶ…еөҢ JavaScript и„ҡжң¬ж Үзӯҫж—¶,HTML и§ЈжһҗеҷЁжҡӮеҒңвјҜдҪңпјҢJavaScript еј•ж“Һд»ӢвјҠпјҢе№¶жү§вҫҸ
    script ж Үзӯҫдёӯзҡ„и„ҡжң¬пјҢ и„ҡжң¬дјҡдҝ®ж”№ DOM дёӯеҶ…е®№пјҢи„ҡжң¬жү§вҫҸе®ҢжҲҗд№ӢеҗҺпјҢHTML и§ЈжһҗеҷЁжҒўеӨҚи§ЈжһҗиҝҮ
    зЁӢпјҢз»§з»ӯи§ЈжһҗеҗҺз»ӯзҡ„еҶ…е®№пјҢзӣҙвҫ„вҪЈжҲҗжңҖз»Ҳзҡ„ DOM

  • JavaScript вҪӮд»¶зҡ„дёӢиҪҪиҝҮзЁӢдјҡйҳ»еЎһ DOM и§ЈжһҗгҖӮ

  • Chrome жөҸи§ҲеҷЁеҒҡдәҶеҫҲеӨҡдјҳеҢ–пјҢе…¶дёӯвјҖдёӘдё»иҰҒзҡ„дјҳеҢ–жҳҜйў„и§Јжһҗж“ҚдҪңгҖӮеҪ“жёІжҹ“еј•ж“Һж”¶еҲ°еӯ—иҠӮжөҒд№ӢеҗҺпјҢ
    дјҡејҖеҗҜвјҖдёӘйў„и§ЈжһҗзәҝзЁӢпјҢвҪӨжқҘеҲҶжһҗ HTML вҪӮд»¶дёӯеҢ…еҗ«зҡ„ JavaScriptгҖҒCSS зӯүзӣёе…івҪӮд»¶пјҢи§ЈжһҗеҲ°зӣё
    е…івҪӮд»¶д№ӢеҗҺпјҢйў„и§ЈжһҗзәҝзЁӢдјҡжҸҗеүҚдёӢиҪҪиҝҷдәӣвҪӮд»¶гҖӮ

  • дҪҝвҪӨ CDN жқҘеҠ йҖҹ JavaScript вҪӮд»¶зҡ„еҠ иҪҪпјҢеҺӢзј© JavaScript вҪӮд»¶зҡ„дҪ“з§ҜгҖӮ

  • еҸҜд»Ҙе°ҶиҜҘ JavaScript и„ҡжң¬и®ҫзҪ®дёәејӮжӯҘеҠ иҪҪпјҢйҖҡиҝҮ async жҲ– defer жқҘж Үи®°д»Јз Ғ

еңЁжү§вҫҸ JavaScript д№ӢеүҚпјҢйңҖиҰҒе…Ҳи§Јжһҗ JavaScript иҜӯеҸҘд№ӢдёҠжүҖжңүзҡ„ CSS ж ·ејҸгҖӮжүҖд»ҘеҰӮжһңд»Јз ҒвҫҘеј•вҪӨдәҶ
еӨ–йғЁзҡ„ CSS вҪӮд»¶пјҢйӮЈд№ҲеңЁжү§вҫҸ JavaScript д№ӢеүҚпјҢиҝҳйңҖиҰҒзӯүеҫ…еӨ–йғЁзҡ„ CSS вҪӮд»¶дёӢиҪҪе®ҢжҲҗпјҢе№¶и§ЈжһҗвҪЈжҲҗ
CSSOM еҜ№иұЎд№ӢеҗҺпјҢжүҚиғҪжү§вҫҸ JavaScript и„ҡжң¬гҖӮ

дёҚз®ЎиҜҘи„ҡжң¬жҳҜеҗҰж“ҚзәөдәҶ CSSOMпјҢйғҪдјҡжү§вҫҸ CSS вҪӮд»¶дёӢиҪҪпјҢи§Јжһҗж“ҚдҪңпјҢеҶҚжү§вҫҸ JavaScript и„ҡжң¬гҖӮ
жҖ»з»“пјҡJavaScript дјҡйҳ»еЎһ DOM вҪЈжҲҗпјҢвҪҪж ·ејҸвҪӮд»¶вјңдјҡйҳ»еЎһ JavaScript зҡ„жү§вҫҸ
